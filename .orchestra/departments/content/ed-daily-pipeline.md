# Ed's Daily Lead Pipeline

_Not a static list. A daily routine that generates fresh leads._

---

## Daily Routine (30-45 min from phone)

### Morning: Product Hunt (10 min) — HIGHEST CONVERSION
These people launched TODAY. They're desperate for distribution.

1. Go to: https://www.producthunt.com/categories/developer-tools
2. Find dev tool launches from today/yesterday
3. Comment on their launch (genuine, not spammy):

> Congrats on the launch! We run IndieStack — AI coding agents (Claude, Cursor) query it to find dev tools. 3,100+ indexed. Yours looks perfect for [category]. Want me to add it? Agents would start recommending it automatically.

4. Then DM the maker on Twitter/LinkedIn with the same offer

### Lunch: GitHub Trending (10 min) — TIME-SENSITIVE
Repos trending NOW want distribution NOW. 48-72 hour window.

1. Check: https://github.com/trending/typescript?since=daily
2. Also: https://github.com/trending/python?since=daily
3. Skip tutorials/awesome-lists. Look for: CLIs, SDKs, frameworks, tools
4. Open an issue on relevant repos:

**Title:** Distribution via AI coding agents?

> Hey — saw [repo] trending, congrats. I run IndieStack (indiestack.ai), a discovery layer that AI agents (Claude, Cursor) query via MCP to find dev tools. 3,100+ indexed. Would you want [tool] listed? Free — means agents start recommending it. Takes 2 min.

### Evening: Indie Hackers (10 min)
Builders actively looking for distribution.

1. Browse: https://www.indiehackers.com/group/what-are-you-working-on
2. Also: https://www.indiehackers.com/products?sorting=newest
3. Reply to anyone building dev tools:

> Nice — [tool] looks solid. We run IndieStack, where AI coding agents discover dev tools via MCP. 3,100+ indexed. Happy to add yours free — agents start recommending it automatically.

### Weekly: LinkedIn Batch (15 min, once a week)
Target DevRel people — partnerships are literally their job.

**Search queries to paste:**
- `"Developer Advocate" AND ("developer tools" OR "devtools")`
- `"DevRel" AND (Resend OR Supabase OR Neon OR Turso OR Drizzle OR Clerk)`
- `"Founder" AND ("developer tools") AND building`

**Connection note (max 300 chars):**
> Hey [name] — [tool] is already on IndieStack, where AI coding agents discover dev tools via MCP. 3,100+ indexed, yours gets recommended. Happy to share your listing stats if useful.

### Bonus: Discord (when bored)
Browse #showcase channels in Claude Code, Cursor, AI Engineer Discord servers.
Rule: react to their post → reply publicly → THEN DM. Never cold DM.

## The Universal Pitch

Adapt for each channel but the core is:

> Your tool + IndieStack = AI agents recommend it to developers automatically.
> 3,100+ tools indexed. 10,000+ installs. Free to list. Takes 2 min.

You're OFFERING free distribution through AI agents. Not asking for anything.

**Key stat to always include: "10,000+ installs from PyPI, 370+ downloads/day."** This is real traction, not a pitch — it's verifiable at pepy.tech/projects/indiestack.

## What NOT to do
- Don't batch-post the same message to multiple subreddits
- Don't cold email (we tried, 0/24 converted)
- Don't DM people on Discord without engaging publicly first
- Don't pitch paid anything — lead with free listing
- Don't overthink the message — short, specific, genuine
